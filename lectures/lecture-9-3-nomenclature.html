Lecture 9.3

Nomenclature and Ligands

Learn the IUPAC workflow for naming coordination compounds and connect ligand denticity to naming choices.

Lecture Index Games

Learning goals

  • Order ligands alphabetically and apply correct prefixes.
  • Distinguish neutral vs anionic ligands in naming.
  • Assign oxidation state and use -ate for anionic complexes.

Lecture flow

1. Identify ligands and denticity.
2. Order ligands alphabetically by name.
3. Apply prefixes and kappa or mu if needed.
4. Name the metal and oxidation state.
5. Append counter ions.

Micro-demo: naming pipeline

Identify ligands: NH3 (ammine) x5, Cl- x1.
Order alphabetically: ammine before chloro.
Metal + oxidation state: cobalt(III) -> pentaamminechlorocobalt(III) chloride.
